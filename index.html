KorOS: An Indie Linux Distro

KorOS is an independent Linux distribution designed to be good to use daily and comparable to bigger Distros of the Linux Family. The KorOS Distro isn't in developement yet, but don't worry! The developement will start soon. any update can be found in the KorOS's News page (link on topbar).
The name KorOS comes from Core, which in my native language means "Heart".

What are the download options?

You can choose between:
• KorOS-Nightly : Early releases that may contain bugs (exists as a branch of the main repo).
• KorOS : Tested releases that are more safe but may be slower (not much slower but it's worth noting).

When will developement start?

The developement will start when i will get a PC as i currently don't have one. For more info you can look at the README in the KorOS repository.
